create-react-app Notes

- npm install -g create-react-app
- create-react-app hello-world
  * creates hello-world directory
  * adds lots of files
  * runs npm install
- cd hello-world
- npm start
  * starts local server
  * opens http://localhost:3000/ in default browser
  * watches for code changes, runs ESLint, rebuilds, and refreshes browser
- npm build
  * creates a minified, production build
